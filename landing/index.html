Instalación

Cómo instalar

En Mac, usa el botón Descargar para macOS arriba. En acceso anticipado, macOS puede mostrar una advertencia de seguridad — es normal con software independiente. Windows (.exe) está en preparación; escríbenos si lo necesitas ya.

  1. Descarga el .zip (Mac) desde el botón de arriba o desde GitHub Releases cuando esté publicado.
  2. Descomprime y arrastra Psicoterapia Lab.app a Aplicaciones.
  3. La primera vez: clic derecho → Abrir (no doble clic directo).
  4. Si macOS bloquea: ve a Ajustes del Sistema → Privacidad y seguridad → Abrir de todas formas.
  5. Configura tu PIN de 6 dígitos y, si tu Mac lo permite, activa Touch ID.
