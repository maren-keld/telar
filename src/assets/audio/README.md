# Audio neurofeedback (local)

Archivos usados por `src/lib/nf-audio.js`:

| Archivo | Uso |
|---------|-----|
| `feedback-relaxation.wav` | Música/ambiente en loop (volumen según bandas) |
| `connected.wav` | Al conectar Muse |
| `low-battery.wav` | Batería ≤ 20% |
| `disconnected.wav` | Desconexión inesperada |

Los actuales son **placeholders** (tonos generados). Puedes sustituirlos por los MP3/WAV originales manteniendo el mismo nombre o actualizando `nf-audio.js`.
