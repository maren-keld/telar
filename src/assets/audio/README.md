# Audio neurofeedback (local)

Archivos usados por `src/lib/nf-audio.js`:

| Archivo | Uso |
|---------|-----|
| `feedback-relaxation.mp3` | Música/ambiente en loop (volumen según bandas) |
| `connected.wav` | Al conectar Muse |
| `low-battery.wav` | Batería ≤ 20% |
| `disconnected.wav` | Desconexión inesperada |

Los tonos de sistema (`connected`, `low-battery`, `disconnected`) siguen en WAV; la pista de feedback usa `feedback-relaxation.mp3`.
